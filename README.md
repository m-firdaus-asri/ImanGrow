# ImanGrow

ImanGrow is a cross-platform mobile app that helps users track daily ibadah (Solat, Quran, and Zikr) with progress and motivation features.

## Features
- Daily tracking for Solat, Quran recitation, and Zikr
- Streak tracking and progress indicators
- Daily Islamic motivations (quotes/verses/hadith)
- Profile and About pages
- Simple multi-page navigation flow

## Tech Stack
- .NET MAUI
- C#
- XAML
- Visual Studio
